## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm start
   ```

3. Run on your preferred platform:
   ```bash
   npm run ios     # for iOS
   npm run android # for Android
   npm run web     # for Web
   ```
